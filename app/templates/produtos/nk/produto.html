O Stroke Coach Surge Rate fornece os dados básicos de um treino de remo, incluindo tempo, voga e contagem de remadas, com memória para consulta, sem o uso de fios.

Speed Coach

O Speed Coach fornece todas as informações necessárias para uma competição, incluindo tempo, voga, distância percorrida e passagem de 500 m.

 

Speed Coach Gold

O famoso Speed Coach Gold contém todas as funções do Speed Coach, com mais espaço de armazenamento, iluminação para o display, e treinos programáveis.

 

Interval 2000

O perfeito relógio para técnicos e timoneiros, com marcações específicas para 2000 metros, consulta imediata e interface para o computador.

 

Speed Coach XL

A supremacia da monitoração de treinos - o Speed Coach XL tem todas as funções do Speed Coach Gold, além de comunicar-se com o técnico para um acompanhamento em tempo real.

 

Cox Box

O sistema Cox-Box mostra o tempo, voga e conta remadas numa tela grande e conta com um sistema de amplificação.

 

Cox Box Mini

Um sistema de amplificação para barcos de competição, a prova d'água e flutuante.

 

Microfone para Timoneiro

Um microfone à porva d'água que é conectado ao Cox Box para amplificação de voz pelos falantes no barco.

 

Falante para Barco
