LogCard

 

Diário de treino virtual

 

Este cartão inteligente é a forma virtual para manter o controle de seu treinamento. Ele capta os dados de seu treino diretamente do monitor de performance. Compatível com o PM3 e PM4, basta inserir o LogCard no monitor antes de seu treino, e o monitor irá armazenar seus dados no LogCard quando terminar o treino.

 

Portátil

Do tamanho de um cartão de crédito, o LogCard se encaixa facilmente em sua carteira para que possa levá-lo junto com você. Especialmente útil se você se exercita em vários locais.

 

Salvar treinamentos favoritos

O monitor de performance permite que você armazene seus cinco treinos favoritos. Você também pode armazenar cinco treinos adicionais em seu LogCard, tornando-o ideal se você tem um monte de exercícios favoritos, ou se você quiser configurá-los rapidamente em uma máquina diferente.

 

Armazenamento de dados para até cinco usuários

Tem mais de uma pessoa para usar a máquina? Não tem problema! Cada LogCard pode armazenar dados para até cinco pessoas, então todos podem compartilhar apenas um LogCard.

 

Transferir dados para o seu diário de treino online

Depois de ter os dados em seu LogCard, você pode usar nosso utilitário Concept2 livre para carregar os dados para o seu Concept2 diário de bordo on-line ou em uma planilha no computador. Basta ligar o monitor de performance ou leitor de cartão ao seu computador e usar o utilitário da Concept2 para gerenciar os dados em seu LogCard.
